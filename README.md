# 비행기 별점 데이터를 활용한 문제점 분석 및 개선 방향 도출

## 📌 프로젝트 개요

비행기 이용자의 별점 데이터를 분석하여 서비스의 문제점을 파악하고, 개선 방향을 도출하기 위해 진행한 데이터 분석 미니프로젝트입니다.

단순히 별점 데이터를 확인하는 것에 그치지 않고, 고객 만족도에 영향을 주는 요소를 분석하여 실제 개선 방향으로 연결하는 것을 목표로 했습니다.

---

## 🎯 프로젝트 목표

- 항공 서비스 별점 데이터를 기반으로 고객 불만 요소 파악
- 데이터 분석을 통해 서비스 개선 방향 도출
- 분석 결과를 문제 해결 관점에서 정리
- 팀원들이 작성한 코드를 하나의 실행 가능한 결과물로 통합

---

## 🙋‍♂️ 나의 역할

**코드 통합 및 최종 실행 검수 담당**

팀원들이 작성한 데이터 처리, 분석, 시각화 관련 코드를 취합하고 전체 실행 흐름에 맞게 정리했습니다.  
또한 실행 중 발생하는 오류를 수정하고, 분석 결과가 문제점과 개선 방향으로 자연스럽게 연결될 수 있도록 전체 흐름을 점검했습니다.

---

- 데이터 분석
- 데이터 시각화

## 🔍 주요 수행 내용

- 팀원들이 작성한 데이터 처리, 분석, 시각화 코드 취합
- 데이터 불러오기, 전처리, 분석, 결과 확인 과정 점검
- 전체 코드 실행 순서 정리
- 변수명 불일치 및 실행 오류 수정
- 최종 제출용 코드 정리
- 분석 결과가 문제점과 개선 방향으로 연결되도록 흐름 점검

---

## ✅ 결과 및 배운 점

이 프로젝트를 통해 데이터 분석은 단순히 결과값을 확인하는 것이 아니라, 분석 결과를 실제 문제점과 개선 방향으로 연결하는 과정이 중요하다는 것을 배웠습니다.

또한 팀 프로젝트에서 여러 사람이 작성한 코드를 하나로 합칠 때, 데이터 흐름과 실행 순서를 정확히 관리하는 것이 최종 결과물의 완성도에 큰 영향을 준다는 점을 경험했습니다.
